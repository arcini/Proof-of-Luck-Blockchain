
### Deployment steps:
- clone this repo
- navigate to this directory and rename the example file `mv env .env`
- `go run main.go`
- open a new terminal window and `nc localhost 9000`
- input a token amount to set stake
- input a BPM 
- wait a few seconds to see which of the two terminals won 
- open as many terminal windows as you like and `nc localhost 9000` and watch Proof of Luck in action!
